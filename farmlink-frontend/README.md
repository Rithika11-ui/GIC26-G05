# FarmLink Frontend 🌱

Nuxt 4 frontend connecting farmers to customers.

## Tech Stack
- Nuxt
- Vue
- Pinia
- Axios
- Tailwind CSS

## Project Setup
```bash
npm install
```

## Environment Setup
```bash
copy .env.example .env
```

Fill in `.env`:
```env
NUXT_PORT=
NUXT_PUBLIC_API_URL=
```

## Compile and Hot-Reload for Development
```bash
npm run dev
```

## Type-Check, Compile and Minify for Production
```bash
npm run build
```

## Preview Production Build
```bash
npm run preview
```
